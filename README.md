# 訪問看護ステーションrecoco ホームページ

このホームページをGitHub Pagesで無料公開する方法を説明します。

## 📋 必要なもの
- GitHubアカウント(無料)
- このフォルダ内のファイル全て

## 🚀 公開手順(約10分)

### ステップ1: GitHubアカウントを作成
1. [GitHub](https://github.com/)にアクセス
2. 右上の「Sign up」をクリック
3. メールアドレス、パスワードを入力して登録
4. メール認証を完了

### ステップ2: 新しいリポジトリを作成
1. GitHubにログイン
2. 右上の「+」マークをクリック → 「New repository」を選択
3. 以下のように入力:
   - **Repository name**: `recoco-hp`(好きな名前でOK)
   - **Public**を選択(無料で公開するため)
   - 「Add a README file」は**チェックしない**
4. 「Create repository」をクリック

### ステップ3: ファイルをアップロード
1. 作成したリポジトリのページで「uploading an existing file」をクリック
2. 以下のファイルを**全て選択**してドラッグ&ドロップ:
   - index.html
   - style.css
   - script.js
   - hero.png
   - README.md
3. 下部の「Commit changes」をクリック

### ステップ4: GitHub Pagesを有効化
1. リポジトリのページで「Settings」タブをクリック
2. 左メニューから「Pages」をクリック
3. 「Source」の下で:
   - **Branch**: `main`を選択
   - **Folder**: `/ (root)`を選択
4. 「Save」をクリック
5. 数分待つと、ページ上部に公開URLが表示されます
   - 例: `https://あなたのユーザー名.github.io/recoco-hp/`

### ステップ5: 確認
- 表示されたURLをクリックしてホームページを確認
- このURLを誰にでも共有できます!

## 🔄 更新方法
ホームページを修正したい場合:
1. GitHubのリポジトリページにアクセス
2. 修正したいファイル(例: index.html)をクリック
3. 鉛筆マーク(Edit)をクリック
4. 内容を修正
5. 「Commit changes」をクリック
6. 数分で自動的に反映されます

## 🌐 独自ドメインの設定(オプション)
`recoco.com`のような独自ドメインを使いたい場合:
1. ドメインを購入(お名前.comなど)
2. GitHub Pagesの設定で「Custom domain」に入力
3. ドメイン側でDNS設定を行う
