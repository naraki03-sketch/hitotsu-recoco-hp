# 📖 ホームページ公開手順(超詳細版)

エンジニアでない方向けの、画像で見るステップバイステップガイドです。

---

## 🎯 この手順でできること
- 訪問看護ステーションrecocoのホームページを**完全無料**で公開
- インターネット上で誰でもアクセスできるようになります
- URL例: `https://あなたの名前.github.io/recoco-hp/`

---

## 📝 事前準備

### 必要なもの
✅ パソコン(Mac/Windows どちらでもOK)  
✅ インターネット接続  
✅ メールアドレス  
✅ このフォルダ内の全てのファイル

### 所要時間
⏱️ 約10〜15分

---

## ステップ1️⃣: GitHubアカウントを作成(5分)

### 1-1. GitHubにアクセス
- ブラウザで https://github.com/ を開く

### 1-2. サインアップ
1. 右上の「**Sign up**」ボタンをクリック
2. 以下を入力:
   - **Email**: あなたのメールアドレス
   - **Password**: 安全なパスワード(8文字以上)
   - **Username**: 好きなユーザー名(例: `recoco-station`)
3. 「**Create account**」をクリック

### 1-3. メール認証
1. 登録したメールアドレスに確認メールが届きます
2. メール内の「**Verify email address**」をクリック
3. これでアカウント作成完了!

---

## ステップ2️⃣: リポジトリを作成(3分)

### 2-1. 新規リポジトリ作成
1. GitHubにログインした状態で、右上の「**+**」マークをクリック
2. 「**New repository**」を選択

### 2-2. リポジトリの設定
以下のように入力してください:

| 項目 | 入力内容 |
|------|----------|
| **Repository name** | `recoco-hp` |
| **Description** | 訪問看護ステーションrecocoのホームページ |
| **Public/Private** | **Public**を選択 ⚠️重要 |
| **Add a README file** | チェック**しない** |

3. 「**Create repository**」ボタンをクリック

---

## ステップ3️⃣: ファイルをアップロード(3分)

### 3-1. アップロード画面を開く
1. 作成したリポジトリのページが表示されます
2. 「**uploading an existing file**」というリンクをクリック
   - または「**Add file**」→「**Upload files**」をクリック

### 3-2. ファイルを選択
1. Finderで`Recoco_HP`フォルダを開く
2. 以下のファイルを**全て選択**:
   ```
   ✅ index.html
   ✅ style.css
   ✅ script.js
   ✅ hero.png
   ✅ README.md
   ✅ 公開手順_画像付き.md
   ```
3. 選択したファイルをブラウザの画面にドラッグ&ドロップ

### 3-3. アップロード実行
1. ファイルがアップロードされるのを待つ
2. 画面下部の「**Commit changes**」ボタンをクリック
3. 数秒でアップロード完了!

---

## ステップ4️⃣: GitHub Pagesを有効化(2分)

### 4-1. 設定画面を開く
1. リポジトリのページで「**Settings**」タブをクリック
   - 画面上部のタブメニューにあります

### 4-2. Pages設定を開く
1. 左側のメニューから「**Pages**」をクリック

### 4-3. 公開設定
1. 「**Source**」セクションで以下を選択:
   - **Branch**: `main` を選択
   - **Folder**: `/ (root)` を選択
2. 「**Save**」ボタンをクリック

### 4-4. 公開完了を確認
1. ページが自動的にリロードされます
2. 数分後、ページ上部に以下のような表示が出ます:
   ```
   ✅ Your site is live at https://ユーザー名.github.io/recoco-hp/
   ```
3. このURLがあなたのホームページのアドレスです!

---

## ステップ5️⃣: 確認とシェア

### 5-1. ホームページを確認
1. 表示されたURL(例: `https://ユーザー名.github.io/recoco-hp/`)をクリック
2. ホームページが表示されることを確認

### 5-2. URLをシェア
- このURLを誰にでも共有できます
- 名刺、チラシ、SNSなどに記載可能
- 世界中どこからでもアクセスできます

---

## 🔧 よくある質問

### Q1: ページが表示されません
**A**: 公開には数分かかります。5〜10分待ってから再度アクセスしてください。

### Q2: 内容を修正したい
**A**: 以下の手順で修正できます:
1. GitHubのリポジトリページにアクセス
2. 修正したいファイル(例: `index.html`)をクリック
3. 鉛筆マーク(✏️)をクリック
4. 内容を修正
5. 「Commit changes」をクリック
6. 数分で自動的に反映されます

### Q3: 独自ドメイン(例: recoco.com)を使いたい
**A**: 
1. ドメインを購入(お名前.com、ムームードメインなど)
2. GitHub Pagesの設定で「Custom domain」に入力
3. ドメイン会社の管理画面でDNS設定を行う
   - 詳細は購入したドメイン会社のサポートを参照

### Q4: 費用はかかりますか?
**A**: GitHub Pagesは完全無料です。独自ドメインを使う場合のみ、ドメイン代(年間1,000〜2,000円程度)がかかります。

### Q5: セキュリティは大丈夫?
**A**: GitHubは世界中で使われている信頼性の高いサービスです。自動的にHTTPS(暗号化)で配信されます。
